## Usages
1. Clone the project into your local repository.
2. Install poetry
3. Install Dependancy   
`
poetry install
`  
4. Init DB  
`
cd code_analysis
`  
`
poetry run flask init-db
`  
5. Run Flask  
`
poetry run flask run
`
